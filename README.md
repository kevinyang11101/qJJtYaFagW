# 企业IT资产管理系统 python1018

## 项目概述

本项目是基于Django框架的Python企业IT资产管理系统，旨在帮助企业高效管理IT资产，涵盖设备、软件、配件及办公耗材等多个方面。

## 技术栈

- 开发语言：Python 3.6.8
- 数据库：MySQL 5.7+
- 数据库工具：Navicat 11+
- 开发环境：PyCharm

## 功能模块

### 系统角色

- 管理员
- 员工

### 功能介绍

- 首页
- 个人中心
- 员工管理
- 设备信息管理
  - 设备借用管理
  - 设备归还管理
- 附属服务管理
- 外包服务管理
- 软件信息管理
  - 软件领用管理
- 配件信息管理
  - 配件领用管理
  - 配件归还管理
- 办公耗材管理
  - 耗材入库管理
  - 耗材领用管理
- 管理员管理
- 系统管理

## 运行环境

- Python 3.6.8
- Django框架
- MySQL 5.7+
- Navicat 11+

（由于涉及具体部署细节，以下仅为概括性说明）

3. 使用Navicat 11+管理数据库
4. 使用PyCharm导入项目源代码
